This project presents a comprehensive retail sales analysis pipeline built on Python, SQL, and Power BI. Raw sales data was first cleaned and normalized using Python, ensuring consistency by handling missing values, correcting formats, and restructuring tables into a relational schema. SQL was then employed to create the Global_Store database, enforce integrity, and implement advanced queries such as monthly sales summaries, product ranking with window functions, and customer segmentation through RFM (Recency, Frequency, Monetary) scoring. Finally, Power BI dashboards transformed these results into interactive visualizations, enabling managers to explore regional performance, top products, and customer value. Together, these three technologies deliver a robust framework for data-driven decision-making in retail.
